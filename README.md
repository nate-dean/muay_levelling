# Muay_Levelling
A game based off of solo levelling designed to help track Muay Thai Progress. It is RPG based so you get experience point in different areas to level up your overall level.

# Features
- Log Muay Thai Training Sessions
- Add Notes on Key take aways
- Earn Experience points in each skill categories
- Session History
- Mobile first - Responsive Design.

# Why this Project?
I love to train Muay Thai and I want a cool way to track progress.
- Reflect on each session.
- Track Skill Progression over time
- Makes training logs more fun through gamification.

# Tech Used
- HTML
- CSS
- JavaScript

# Responsive Design
Muay Levelling is built with the idea of it being primarily used on a mobile device for ease to log after training. While also scaling to tablet and desktop screens sizes.
